Définition

SQL (Structured Query Language) est un langage informatique utilisé pour communiquer avec les bases de données relationnelles. Il permet de créer, modifier et interroger des données stockées dans ces bases de données. Le SQL est utilisé dans de nombreux contextes professionnels tels que le développement de logiciels, la gestion de bases de données, la business intelligence, le marketing et l'analyse de données.

Mon expérience

Dans tous les projets sur lesquels j'ai travaillé jusqu'à présent, il a été nécessaire de rendre les données persistantes.

Pour les projets PopOff et Bandngine, les informations de connexion des utilisateurs étaient stockées dans la base de données, bien sûr chiffrées, pour assurer la sécurité. Mais plus que juste les comptes d'utilisateurs, des "fiches" d'artistes et de mouvements de danse étaient également stockées, permettant de retrouver facilement des informations de contact ou les étapes pour exécuter un mouvement spécifique.
Les données stockées étaient ensuite récupérées à l'aide de commandes SQL et affichées à l'utilisateur lorsqu'il souhaitait les consulter.

Pour le projet Tryat, la base de données était plus simple. Elle avait pour unique but de stocker les données traitées dans le backend après leur réception via l'API. Ensuite, ces données étaient affichées sur la page d'accueil du site dans un tableau.

Enfin, Thalys utilise une base de données un peu spéciale et complexe à prendre en main. En effet, Drupal stocke énormément d'informations, qui sont parfois difficiles à interpréter pour un être humain. Cela permet de conserver toutes les créations de contenu réalisées par le client. Lorsqu'une personne visite une page du site, toutes les informations du contenu de cette page se trouvent dans la base de données. Il est donc conseillé d'utiliser Drupal avec un système de cache pour éviter de solliciter excessivement la base de données.

Ensuite ?

Avec l'explosion de la quantité de données à traiter dans le monde professionnel, la maîtrise du SQL est devenue une compétence de plus en plus importante pour de nombreux postes. Les entreprises recherchent des candidats capables de gérer et de créer des bases de données, d'analyser les informations stockées et de produire des rapports sur ces données. Cette compétence est particulièrement vitale pour les professionnels travaillant dans des secteurs tels que la finance, la santé, la technologie, ainsi que dans de nombreux autres domaines, en raison de l'essor du Big Data et de l'analyse prédictive. En conséquence, la capacité à utiliser le SQL est devenue un atout majeur pour les candidats cherchant à se démarquer sur le marché de l'emploi.
