Définition

Ce que l'on sous entend par la base web est un ensemble de technologie utilisé pour créer un site web. Cela prend en compte html, javascript (js), css et php. Un très grande majorité des sites web d'aujourd'hui sont composés de ces technologies, ils sont omniprésents. Dès que vous visitez un site web, vous pouvez être sûr qu'il est composé toute ou parti de cette base.

Mon expérience

Par exemple, le site de Thalys est fait à base de php, html, css et js. Pour être plus précis, le formulaire de réservation de billet fonctionne grâce à php et js. L'affichage du formulaire est complétement fait sur base php, plus précisément, c'est du Drupal (framework), et le fonctionnement du bouton d'échange de gare de départ/arrivé est géré en js. Son effet de rotation est un mélange de js qui lui donne une classe css pour appliquer l'effet de mouvement. C'est d'ailleurs un fonctionnalité que j'ai développé, c'est pour cela que je peux vous décrire la manière dont c'est fait.

Ensuite ?

Je travaille maintenant depuis plus de 5 ans sur ces technologies de manière quotidienne donc je pense pouvoir dire que j'ai une certaine maîtrise sur ces sujets. Ce qui me permet de vite identifier si ces technologies sont adaptées ou non à une demande et d'apporter mon aide en cas de besoin. Si je peux donner un conseil à quelqu'un qui débute sur ces technologies, c'est de suivre les documentations et formation sur Internet, notamment sur w3school.

Pour les mois à venir, je vais être sur le projet Thalys qui fonctionne principalement grâce à cette base, la difficulté des fonctionnalités qui vont devoir être développé reste encore à déterminer.
