Définition

La base web fait référence à un ensemble de technologies utilisées pour créer des sites web dynamiques et interactifs. Ces technologies comprennent :

  • HTML (Hypertext Markup Language) : Il s'agit du langage de balisage utilisé pour structurer le contenu d'une page web, tels que les textes, les images, les vidéos, les liens, etc.
  • CSS (Cascading Style Sheets) : Il est utilisé pour styliser la présentation de la page web en définissant les couleurs, les polices, les mises en page, les animations, etc.
  • JavaScript : C'est un langage de programmation qui permet d'ajouter des fonctionnalités interactives à une page web, telles que des menus déroulants, des formulaires interactifs, des animations, des effets visuels, etc.
  • PHP (Hypertext Preprocessor) : Il s'agit d'un langage de programmation côté serveur qui permet de traiter les données du site web avant de les envoyer au navigateur du visiteur. Il est souvent utilisé pour créer des sites web dynamiques capables de stocker et récupérer des données à partir d'une base de données.

En combinant ces quatre technologies, les développeurs web peuvent créer des sites web riches en fonctionnalités et en contenu, offrant ainsi une expérience utilisateur agréable et interactive.

Mon expérience

Par exemple, le site Thalys que je suis en train de développer utilise ces technologies de base. Plus précisément, le formulaire de réservation de billets fonctionne grâce à PHP et JavaScript. L'affichage du formulaire est entièrement basé sur PHP, en utilisant le framework Drupal, tandis que le fonctionnement du bouton d'échange de gare de départ/arrivée est géré en JavaScript. L'effet de rotation est réalisé en combinant JavaScript, qui ajoute une classe CSS pour appliquer l'effet de mouvement. J'ai d'ailleurs développé cette fonctionnalité moi-même, donc je peux vous décrire en détail sa mise en œuvre.

Ensuite ?

Après plus de 5 ans de travail quotidien sur ces technologies de base, je peux dire que j'ai acquis une certaine maîtrise dans ce domaine. Cela me permet de rapidement évaluer si ces technologies conviennent ou non à une demande, et d'apporter mon aide en cas de besoin. Si je peux donner un conseil à quelqu'un qui débute dans ces technologies, c'est de suivre les documentations et les formations disponibles en ligne, notamment sur des sites comme W3Schools.

Pour les mois à venir, je vais continuer à travailler sur le projet Thalys, qui repose principalement sur cette base technologique. La complexité des fonctionnalités à développer reste encore à déterminer.
