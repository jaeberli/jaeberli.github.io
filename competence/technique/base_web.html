Définition

La base web se réfère à un ensemble de technologies utilisées pour créer des sites web dynamiques et interactifs.

  • HTML (Hypertext Markup Language) est le langage de balisage utilisé pour structurer le contenu de la page web, tels que les textes, les images, les vidéos, les liens, etc.
  • CSS (Cascading Style Sheets) est utilisé pour styliser la présentation de la page web en définissant les couleurs, les polices, les mises en page, les animations, etc.
  • JavaScript est un langage de programmation qui permet d'ajouter des fonctionnalités interactives à une page web, tels que des menus déroulants, des formulaires interactifs, des animations, des effets visuels, etc.
  • PHP (Hypertext Preprocessor) est un langage de programmation côté serveur qui permet de traiter les données du site web avant de les envoyer au navigateur du visiteur. Il est souvent utilisé pour créer des sites web dynamiques qui peuvent stocker et récupérer des données à partir d'une base de données.
En combinant ces quatre technologies, les développeurs web peuvent créer des sites web riches en fonctionnalités et en contenu qui sont attractifs pour les visiteurs et offrent une expérience utilisateur agréable.

Mon expérience

Par exemple, le site de Thalys est fait à base de php, html, css et js. Pour être plus précis, le formulaire de réservation de billet fonctionne grâce à php et js. L'affichage du formulaire est complétement fait sur base php, plus précisément, c'est du Drupal (framework), et le fonctionnement du bouton d'échange de gare de départ/arrivé est géré en js. Son effet de rotation est un mélange de js qui lui donne une classe css pour appliquer l'effet de mouvement. C'est d'ailleurs un fonctionnalitée que j'ai développée, c'est pour cela que je peux vous décrire la manière dont s'est fait.
