Définition

Un framework est une plateforme logicielle préconstruite qui fournit des outils et des bibliothèques pour faciliter le développement de logiciels et accélérer le processus de programmation. Il permet aux développeurs de se concentrer sur les aspects uniques de leur application plutôt que de réinventer la roue pour des fonctions de base. Les frameworks sont utilisés dans de nombreux langages de programmation tels que JS, PHP, Python, Java et bien d'autres, et offrent une architecture de base pour les applications web, les applications mobiles, les logiciels de bureau et d'autres types de logiciels.

Mon expérience

J'ai eu l'occasion de travailler avec plusieurs frameworks depuis mon BTS. La première fois, c'était pour le projet Tryat. Ce projet constituait la base de notre oral final, et nous devions le développer rapidement pour obtenir des résultats concrets dans les délais impartis et pouvoir tester toutes les connexions via l'API de la balise SIGFOX. Pour ce projet, nous avons utilisé Symfony, un framework très répandu et considéré comme le "père" des CMS (Content Management Systems). Mon rôle était de mettre en place le back-end en suivant l'architecture Modèle-Vue-Contrôleur (MVC), tandis qu'un de mes collègues s'occupait de la partie front-end (Vue) du site. Grâce à l'utilisation de Symfony, nous avons pu développer rapidement un site web fonctionnel avec une gestion efficace des données et une expérience utilisateur optimale.

J'ai également eu l'occasion de travailler avec deux frameworks simultanément sur le projet Thalys. Tout d'abord, React, un framework JavaScript en plein essor. Nous avons utilisé React pour développer la partie compte utilisateur du site web Thalys. Son principal avantage était la possibilité d'avoir une seule page dont le contenu changeait dynamiquement, évitant ainsi les redirections ou les chargements de page qui auraient pu nuire à l'expérience utilisateur. J'ai contribué au développement de cette fonctionnalité en utilisant React et en intégrant les API nécessaires pour la gestion des comptes utilisateurs. De plus, j'ai utilisé Drupal, un CMS basé sur Symfony, pour proposer un back-office permettant d'ajouter du contenu à un site web. J'ai personnalisé les formulaires et leur logique en utilisant les fonctionnalités flexibles de Drupal. Ces deux frameworks m'ont permis de développer des fonctionnalités avancées de manière efficace et d'obtenir des résultats concrets pour le projet Thalys.

Ensuite ?

Dans le contexte professionnel actuel, les frameworks sont devenus des éléments clés du développement logiciel. Ils sont utilisés pour développer rapidement des applications de haute qualité, avec une structure solide et un code maintenable. Les frameworks sont particulièrement utiles pour les entreprises qui cherchent à développer des applications de manière agile, en réduisant les délais de mise sur le marché et en optimisant les processus de développement. Ils sont également essentiels dans des domaines émergents tels que l'Internet des objets (IoT) et l'intelligence artificielle (IA), où ils permettent aux développeurs de se concentrer sur des fonctionnalités spécifiques plutôt que de se préoccuper des détails techniques. En tant que développeur, je considère la maîtrise des frameworks comme une compétence clé dans mon profil. J'ai acquis une solide expérience dans l'utilisation de Symfony, React et Drupal, et j'ai la capacité de les mettre en œuvre pour développer des applications de qualité. Cependant, je reste ouvert à l'apprentissage de nouveaux frameworks pour élargir mes compétences et rester à jour avec les avancées technologiques. À moyen terme, je vise à devenir un expert dans l'utilisation de plusieurs frameworks et à les utiliser de manière optimale pour des projets variés.

Actuellement, je suis engagé dans une formation en ligne axée sur le développement avec Angular, un autre framework JavaScript populaire. Cette formation me permettra d'élargir mes compétences dans le domaine des frameworks et d'acquérir une compréhension approfondie d'Angular. J'ai également prévu de suivre des cours sur Vue.js, un autre framework JavaScript très demandé. Je considère ces formations comme un investissement dans ma carrière professionnelle, car elles me permettront de répondre aux besoins du marché et de travailler sur des projets variés nécessitant différentes compétences en matière de frameworks. Je suis motivé pour continuer à développer mes compétences et à m'adapter aux évolutions technologiques rapides du secteur du développement logiciel.
