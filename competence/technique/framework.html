Définition

Un framework est une plateforme logicielle préconstruite qui fournit des outils et des bibliothèques pour faciliter le développement de logiciels et accélérer le processus de programmation. Il permet aux développeurs de se concentrer sur les aspects uniques de leur application plutôt que de réinventer la roue pour des fonctions de base. Les frameworks sont utilisés dans de nombreux langages de programmation tels que JS, PHP, Python, Java et bien d'autres, et offrent une architecture de base pour les applications web, les applications mobiles, les logiciels de bureau et d'autres types de logiciels.

Mon expérience

J'ai eu l'occasion de travailler avec plusieurs frameworks depuis mon BTS. La première fois, c'était pour le projet Tryat. Ce projet constituait la base de notre oral final, et nous devions le développer rapidement pour obtenir rapidement des résultats concrets et pouvoir tester toutes les connexions via l'API de la balise SIGFOX. Nous avons utilisé un framework très répandu et considéré comme le "père" des CMS (Content Management Systems) : Symfony. L'architecture Modèle-Vue-Contrôleur a facilité la prise en main, avec un de mes collègues s'occupant de la partie front-end (Vue) du site et moi de la mise en place du back-end (Modèle, Contrôleur).

J'ai également eu l'occasion de travailler avec deux frameworks simultanément sur le projet Thalys. Tout d'abord, React, un framework JavaScript en plein essor. Son principal avantage est la possibilité d'avoir une seule page dont le contenu change dynamiquement, évitant ainsi les redirections ou les chargements de page qui pourraient nuire à l'expérience utilisateur si cela prend trop de temps. La partie compte utilisateur du site web Thalys a été entièrement développée en React. Bien que la prise en main puisse sembler complexe, il existe des tutoriels très bien conçus pour comprendre les mécanismes de ce framework. Ensuite, j'ai utilisé Drupal, un CMS basé sur Symfony, qui permet de proposer un back-office permettant d'ajouter du contenu à un site web. Drupal offre une grande flexibilité, car les formulaires sont en réalité des tableaux (arrays), ce qui permet de les modifier selon les besoins.

Ensuite ?

Dans le contexte professionnel actuel, les frameworks sont devenus des éléments clés du développement logiciel. Ils sont utilisés pour développer rapidement des applications de haute qualité, avec une structure solide et un code maintenable. Les frameworks sont particulièrement utiles pour les entreprises qui cherchent à développer des applications de manière agile, en réduisant les délais de mise sur le marché et en optimisant les processus de développement. Les frameworks sont également de plus en plus utilisés dans les technologies émergentes telles que l'Internet des objets (IoT) et l'intelligence artificielle (IA), où ils permettent aux développeurs de travailler sur des projets plus complexes en se concentrant sur des fonctionnalités spécifiques plutôt que de se préoccuper des détails techniques. Les frameworks sont donc une compétence clé pour les développeurs et les ingénieurs logiciels souhaitant travailler efficacement dans l'industrie du développement logiciel.
